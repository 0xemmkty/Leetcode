# 50. Pow(x, n)

Implement pow(x, n), which calculates x raised to the power n (i.e., xn).

## Solution 1 (分治角度 快速幂)
```java
class Solution {
    public double myPow(double x, int n) {
        if(x == 0.0f) return 0.0d;
        long b = n;
        double res = 1.0;
        if(b < 0) {
            x = 1 / x;
            b = -b;
        }
        while(b > 0) {
            if((b & 1) == 1) res *= x;
            x *= x;
            b >>= 1;
        }
        return res;
    }
}
```
